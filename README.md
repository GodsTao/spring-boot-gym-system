# spring-boot-gym-system（体育馆系统）

## 关于项目
使用spring-boot框架完成的体育馆管理系统，是本人的一次课程实贱，主要有后台管理，器材管理，场地管理，赛事管理等功能，我目前仅完成了后台用户管理的功能

## 技术栈

* spring-boot 核心框架
* mybatis 持久化框架
* shiro  权限框架

* Jquery   js框架
* Bootstrap UI框架
